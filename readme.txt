OBJETIVOS DEL PROYECTO
    • Recrear los diseños proporcionados.
    • Aplicar correctamente el tipo de display necesario de cada diseño y priorizar 
    su visualización en diferentes tamaños de dispositivos manteniendo un diseño 
    coherente y atractivo.

 Requisitos
    Estructura HTML y CSS
        • Usa una estructura HTML bien organizada y semántica.
        • Mantén tu archivo CSS en un archivo separado y enlazado correctamente al 
        HTML.
        • El CSS debe ser anidado, empleando Selectores CSS hijos y descendientes 
        para organizar el estilo de los componentes.
        •Uso de reset CSS.
        •Utilización de variables para estilos repetitivos.
        • No modifiques el reset; este archivo debe mantenerse intacto.
        •IMPRESCINDIBLE RESPONSIVE.
    Componentes Estilizados
        • Estiliza los diferentes componentes de la página, como la cabecera, el 
        cuerpo, las secciones, el pie de página, etc. Utiliza propiedades como flexbox o
        grid para organizar el contenido de manera similar a la página original.
    Estructura de Archivos
        •index.html: Archivo principal con la estructura HTML de la página.
        •styles/ reset.css: Archivo para los resets de CSS. 
        •main.css: Archivo principal de estilos donde deben ir todos los estilos personalizados y anidados.
        •styles/ variables.css: Archivo para las variables CSS.
        •styles/ main.css: Archivo de estilos generales.
